# Introduction Notes

## What is Python?

Python is a high-level, interpreted, general-purpose programming language.

It is widely used in:

- AI
- Machine Learning
- Data Science
- Automation
- Backend Development
- Web Development

---

## Why Python?

- Easy syntax
- Readable code
- Huge community
- Large number of libraries
- Cross-platform
- Beginner friendly

---

## Python Interpreter

Python executes code line by line using the Python Interpreter.

Unlike C++, Python does not need compilation before execution.

---

## Interactive Mode

Interactive mode is opened by typing:

python

or

python3

You will see:

>>>

This means Python is ready to execute commands.

Example:

>>> 2 + 2

Output:

4

---

## Script Mode

Instead of writing code inside the interpreter, we usually create a file.

Example:

hello.py

Then run:

python hello.py

This is called Script Mode.

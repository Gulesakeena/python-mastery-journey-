# Virtual Environment Interview Questions

## Theory

1. What is a virtual environment?
2. Why do we use virtual environments?
3. Difference between pip and venv?
4. What is requirements.txt?
5. What does `pip freeze` do?
6. Why should `.venv` not be pushed to GitHub?
7. How do you activate a virtual environment?
8. How do you deactivate it?
9. What happens if you don't use virtual environments?
10. How do you recreate a project's environment on another computer?

---

## Practical Tasks

1. Create a virtual environment.
2. Activate it.
3. Install NumPy and Pandas.
4. Generate a `requirements.txt`.
5. Delete the environment.
6. Recreate it.
7. Install dependencies from `requirements.txt`.
8. Verify installed packages using `pip list`.

# Virtual Environment Commands

## Create

```bash
python -m venv .venv
```

---

## Activate (Windows PowerShell)

```powershell
.venv\Scripts\Activate
```

---

## Install Packages

```bash
pip install requests
pip install numpy
pip install pandas
```

---

## Show Installed Packages

```bash
pip list
```

---

## Save Packages

```bash
pip freeze > requirements.txt
```

---

## Install Packages

```bash
pip install -r requirements.txt
```

---

## Deactivate

```bash
deactivate
```

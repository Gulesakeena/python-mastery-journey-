# Python Output Notes

## What is Output?

Output is the information displayed to the user.

In Python, the `print()` function is used to display output.

Example:

```python
print("Hello, World!")
```

Output:

```
Hello, World!
```

---

## Printing Text

Text is written inside quotes.

Example:

```python
print("Python")
```

---

## Printing Numbers

Numbers do not need quotes.

Example:

```python
print(100)
```

---

## Printing Expressions

Python evaluates expressions before printing them.

Example:

```python
print(10 + 5)
```

Output:

```
15
```

---

## Printing Multiple Values

You can print more than one value.

Example:

```python
print("Age:", 21)
```

Output:

```
Age: 21
```

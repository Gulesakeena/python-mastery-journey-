# 22 - Python Dates

## Objective

Learn how to work with dates and times in Python using the built-in `datetime` module.

---

# Topics Covered

- Introduction to the `datetime` module
- Getting the current date and time
- Creating custom dates
- Accessing date and time components
- Formatting dates with `strftime()`
- Converting strings to dates with `strptime()`
- Date calculations using `timedelta`
- Finding the difference between two dates
- Common interview questions
- Best practices

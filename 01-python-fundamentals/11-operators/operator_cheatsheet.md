# ⚡ Python Operators Cheat Sheet

> Quick reference for all Python operators.

---

# Arithmetic Operators

| Operator | Description | Example | Result |
|----------|-------------|---------|--------|
| `+` | Addition | `5 + 2` | `7` |
| `-` | Subtraction | `5 - 2` | `3` |
| `*` | Multiplication | `5 * 2` | `10` |
| `/` | Division | `5 / 2` | `2.5` |
| `//` | Floor Division | `5 // 2` | `2` |
| `%` | Modulus | `5 % 2` | `1` |
| `**` | Exponent | `5 ** 2` | `25` |

---

# Assignment Operators

| Operator | Example | Same As |
|----------|---------|---------|
| `=` | `x = 5` | Assign |
| `+=` | `x += 2` | `x = x + 2` |
| `-=` | `x -= 2` | `x = x - 2` |
| `*=` | `x *= 2` | `x = x * 2` |
| `/=` | `x /= 2` | `x = x / 2` |
| `//=` | `x //= 2` | `x = x // 2` |
| `%=` | `x %= 2` | `x = x % 2` |
| `**=` | `x **= 2` | `x = x ** 2` |

---

# Comparison Operators

| Operator | Meaning |
|----------|---------|
| `==` | Equal |
| `!=` | Not Equal |
| `>` | Greater Than |
| `<` | Less Than |
| `>=` | Greater Than or Equal |
| `<=` | Less Than or Equal |

Example

```python
10 > 5
```

Output

```python
True
```

---

# Logical Operators

| Operator | Meaning |
|----------|---------|
| `and` | Both conditions must be True |
| `or` | At least one condition is True |
| `not` | Reverse Boolean value |

Example

```python
True and False
```

Output

```python
False
```

---

# Identity Operators

| Operator | Meaning |
|----------|---------|
| `is` | Same object |
| `is not` | Different object |

a = [1]
b = a

# 29 - Object-Oriented Programming (OOP) Introduction

## 🎯 Objective

Understand the fundamental concepts of Object-Oriented Programming (OOP), why it exists, and why it is widely used in modern software development.

This module introduces the theory behind OOP before learning how to create classes and objects.

---

## 📚 Topics Covered

- What is Object-Oriented Programming?
- Why OOP was introduced
- Procedural Programming vs OOP
- What are Classes?
- What are Objects?
- Attributes and Methods
- Real-world Examples
- Advantages of OOP
- Disadvantages of OOP
- Applications of OOP

## 🎯 Learning Outcomes

After completing this module, you should be able to:

- Explain OOP in your own words.
- Understand why OOP is important.
- Differentiate between procedural programming and OOP.
- Identify classes, objects, attributes, and methods in real life.
- Prepare for learning Classes and Objects in the next module.
